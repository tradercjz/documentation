# demean

## 语法

`demean(X)`

## 参数

**X** 数值型标量、向量、矩阵或表。

## 详情

对一组数据进行去均值化（零均值化）操作，空值不参与计算，返回一个和 *X* 维度相同的 DOUBLE 类型对象。

* 若 *X* 为向量，计算 X - avg(X)；
* 若 *X* 为矩阵，则在矩阵每一列进行计算；
* 若 *X* 为表，则只对数值列进行计算。

## 例子

```
x = 1 NULL 2 3
demean(x)
```

输出返回：[-1,,0,1]

```
v = 1 0 1 1 8 2 -4 0
demean(v)
```

输出返回：[-0.125,-1.125,-0.125,-0.125,6.875,0.875,-5.125,-1.125]
