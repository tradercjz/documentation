# det

## 语法

`det(X)`

## 参数

**X** 是一个矩阵。

## 详情

返回矩阵 *X* 的行列式（determinant）。在计算中，NULL 值用 0 代替。

## 例子

```
x=1..4$2:2;
x;
```

| #0 | #1 |
| --- | --- |
| 1 | 3 |
| 2 | 4 |

```
x.det();
// output
-2

x=1 2 3 6 5 4 8 7 0$3:3;
x;
```

| #0 | #1 | #2 |
| --- | --- | --- |
| 1 | 6 | 8 |
| 2 | 5 | 7 |
| 3 | 4 | 0 |

```
det(x);
// output
42

x=1 2 3 6 5 4 8 7 NULL $3:3;
x;
```

| #0 | #1 | #2 |
| --- | --- | --- |
| 1 | 6 | 8 |
| 2 | 5 | 7 |
| 3 | 4 |  |

```
det(x);
// output
42
```
