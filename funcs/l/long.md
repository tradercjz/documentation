# long

## 语法

`long(X)`

## 详情

将 *X* 的数据类型转换为 LONG。

## 参数

**X** 可以是任意数据类型。

## 例子

```
x=long();
x;
```

返回：null

```
typestr x;
```

返回：LONG

```
long(`10.9);
```

返回：10

```
long(9223372036854775807l);
```

返回：9,223,372,036,854,775,807

注： LONG 数据类型的最大值是263-1 =
9,223,372,036,854,775,807。

```
long(9223372036854775808l);
```

返回：9,223,372,036,854,775,807
