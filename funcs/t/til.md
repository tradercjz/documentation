# til

## 语法

`til(n)`

## 参数

**n** 非负整数。

## 详情

返回 0 到 *n*-1 的整型向量。注意：若 *n* 为 0，则返回空向量。

注：

返回值的类型与 *n* 的类型保持一致，如：若 *n* 类型是 LONG，则返回值的类型是 FAST
LONG VECTOR。

## 例子

```
til(0);
// output
[]

til(5);
// output
[0,1,2,3,4]

n = 10;
t = table(2022.01.01 + til(n) as date, rand(10.0, n) as val);
t;
// output
date        val
2022.01.01  8.403
2022.01.02  9.424
2022.01.03  0.4779
2022.01.04  1.8934
2022.01.05  9.6637
2022.01.06  1.7993
2022.01.07  7.1143
2022.01.08  8.3044
2022.01.09  2.6919
2022.01.10  1.9294
```
