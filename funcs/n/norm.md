# norm

## 语法

`norm(mean, std, count)`

## 参数

**mean** 数值型标量，表示正态分布的期望值。

**std** 数值型标量，表示正态分布的标准差。

**count** 整型标量或数据对。若为标量，表示输出向量的长度；若为数据对，表示输出矩阵的维度。

## 详情

返回一个长度（维度）为 *count* 的向量（矩阵），服从期望值为 *mean*，标准差为 *std* 的正态分布。

## 例子

```
norm(2.0,0.1,3);
// output
[2.026602,1.988621,2.101107]

mean norm(3,1,10000);
// output
3.007866

std norm(3,1,10000);
// output
0.995806

// 生成随机矩阵
norm(0, 1, 3:2)
```
