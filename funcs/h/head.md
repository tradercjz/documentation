# head

## 语法

`head(X, [n=1])`

## 参数

**X** 可以是向量、矩阵或表。

**n** 是一个正整数。

## 详情

`head` 返回向量的前 *n* 个元素，或矩阵的前 *n* 列，或表的前
*n* 行。如果 *n* 没有指定，*n* 取默认值1。

## 例子

```
x=1..10; head(x);
// output
1

x=1..10$2:5;
x;
```

| #0 | #1 | #2 | #3 | #4 |
| --- | --- | --- | --- | --- |
| 1 | 3 | 5 | 7 | 9 |
| 2 | 4 | 6 | 8 | 10 |

```
x.head();
// output
[1,2]

head(x,2);
```

| #0 | #1 |
| --- | --- |
| 1 | 3 |
| 2 | 4 |

```
x=table(1..5 as a, 6..10 as b);
x;
```

| a | b |
| --- | --- |
| 1 | 6 |
| 2 | 7 |
| 3 | 8 |
| 4 | 9 |
| 5 | 10 |

```
head(x);
// output
b->6
a->1

x.head(2);
```

| a | b |
| --- | --- |
| 1 | 6 |
| 2 | 7 |
