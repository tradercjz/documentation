# getJobMessage

## 语法

`getJobMessage(jobId)`

## 参数

**jobId** 是批处理作业的 ID，是一个字符串。

## 详情

取得批处理任务的中间信息。详情请参考 [BatchJobManagement](../../sys_man/BatchJobManagement.html)。

## 例子

```
def job1(n){
   s = 0
   for (x in 1 : n) {
       s += sum(sin rand(1.0, 100000000)-0.5)
       print("iteration " + x + " " + s)
   }
   return s
}
job1_ID=submitJob("job1_ID","", job1, 100);
```

批处理作业完成后，执行以下命令：

```
getJobMessage(job1_ID);
```
