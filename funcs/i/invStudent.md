# invStudent

## 语法

`invStudent(df, X)`

## 参数

**df** 是正数，表示t分布的自由度。

**X** 是0到1之间的浮点型标量或向量。

## 详情

返回t分布的累计密度函数的逆函数值。

## 例子

```
invStudent(1, [0.15, 0.25, 0.35]);
// output
[-1.962611, -1, -0.509525]

invStudent(1, [0.1, 0.3, 0.5, 0.7, 0.9]);
// output
[-3.077684, -0.726543, 0, 0.726543, 3.077684]
```
