# invChiSquare

## 语法

`invChiSquare(df, X)`

## 参数

**df** 是正数，表示卡方分布的自由度。

**X** 是0到1之间的浮点型标量或向量。

## 详情

返回卡方分布的累计密度函数的逆函数值。

## 例子

```
invChiSquare(1, [0, 0.05, 0.15, 0.25]);
// output
[0, 0.003932, 0.035766, 0.101531]

invChiSquare(1, [0.1, 0.3, 0.5, 0.7, 0.9]);

// output
[0.015791, 0.148472, 0.454936, 1.074194, 2.705543]
```
