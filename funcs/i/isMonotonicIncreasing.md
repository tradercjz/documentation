# isMonotonicIncreasing

## 语法

`isMonotonicIncreasing(X)`

## 参数

**X** 可以是标量或向量。

## 详情

判断 *X* 是否为单调递增。

## 例子

```
a=[int(),2,5,7,10]
isMonotonicIncreasing(a);
// output
true

a=[2.1,double(),3.5,4.7,8.2,10.5]
isMonotonicIncreasing(a);
// output
false

a=[5,10,14,20,int()]
isMonotonicIncreasing(a);
// output
false
```
