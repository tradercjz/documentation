# invNormal

## 语法

`invNormal(mean, stdev, X)`

## 参数

**mean** 是正态分布的均值。

**stdev** 是正态分布的标准差。

**X** 是0到1之间的浮点型标量或向量。

## 详情

返回均值为 *mean*，标准差为 *stdev* 的正态分布的累计密度函数的逆函数值。

## 例子

```
invNormal(0,1,0.33);
// output
-0.439913

invNormal(10, 20, [0.1, 0.2, 0.3]);
// output
[-15.631031, -6.832425, -0.48801]
```
