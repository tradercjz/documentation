# int

## 语法

`int(X)`

## 详情

将输入的数据转换为 INT 类型数据。

## 参数

**X** 可以是任意数据类型。

## 例子

```
x=int();
x;
```

返回：null

```
typestr x;
```

返回：INT

```
int(`10.9);
```

返回：10

```
int(2147483647);
```

返回：2,147,483,647

注： INT 数据类型的最大值为 231 -1 = 2,147,483,647。

```
int(2147483648);
```

由于2,147,483,648 超出了 INT 数据类型的最大值，因此返回：null
