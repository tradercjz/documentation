# invGamma

## 语法

`invGamma(shape, scale, X)`

## 参数

**shape** 为形状参数，是正数。

**scale** 为尺度参数，是正数。

**X** 是0到1之间的浮点型标量或向量。

## 详情

返回Gamma分布的累计密度函数的逆函数值。

## 例子

```
invGamma(2.31, 0.627, [0.001, 0.5, 0.999]);
// output
[0.049713, 1.245583, 6.191955]

invGamma(2.31,0.627, [0.1, 0.3, 0.5, 0.7, 0.9]);
// output
[0.437696, 0.843572, 1.245583, 1.760732, 2.724121]
```
