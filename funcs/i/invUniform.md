# invUniform

## 语法

`invUniform(lower, upper, X)`

## 参数

**lower** 和 **upper** 是数值型标量，表示连续均匀分布的下限和上限。

**X** 是0到1之间的浮点型标量或向量。

## 详情

返回均匀分布的累计密度函数的逆函数值。

## 例子

```
invUniform(0.627, 2.31, [0.001, 0.5, 0.999]);
// output
[0.628683, 1.4685, 2.308317]

invUniform(0.627, 2.31, [0.1, 0.3, 0.5, 0.7, 0.9]);
// output
[0.7953, 1.1319, 1.4685, 1.8051, 2.1417]
```
