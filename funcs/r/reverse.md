# reverse

## 语法

`reverse(X)`

## 参数

**X** 可以是向量、矩阵。

## 详情

返回一个新的向量或矩阵，它和原有向量或矩阵的元素顺序相反

## 例子

```
reverse `hello `world;
// output
[world,hello]

(1..6).reverse();
// output
[6,5,4,3,2,1]
x=1..6$2:3;
x
```

| #0 | #1 | #2 |
| --- | --- | --- |
| 1 | 3 | 5 |
| 2 | 4 | 6 |

```
reverse(x);
```

| #0 | #1 | #2 |
| --- | --- | --- |
| 6 | 4 | 2 |
| 5 | 3 | 1 |
