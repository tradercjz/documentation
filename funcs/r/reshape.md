# reshape

## 语法

`reshape(obj, [dim])`

## 参数

**obj** 是一个向量或矩阵。

**dim** 是一个数据对，表示返回结果的行数和列数。它是一个可选参数。

## 详情

返回一个改变形状后的矩阵。如果没有指定 *dim*，则将 *obj* 重组为一个向量。

## 例子

```
x=1..6;
x=x.reshape(3:2);
x
```

| #0 | #1 |
| --- | --- |
| 1 | 4 |
| 2 | 5 |
| 3 | 6 |

```
x=x.reshape(2:3);
x
```

| #0 | #1 | #2 |
| --- | --- | --- |
| 1 | 3 | 5 |
| 1 | 4 | 6 |

```
x=x.reshape(6:1)
x
```

| #0 |
| --- |
| 1 |
| 2 |
| 3 |
| 4 |
| 5 |
| 6 |

```
x.reshape()
// output
[1,2,3,4,5,6]    // 将 x 重组为一个向量
```
