# ratios

## 语法

`ratios(X)`

## 参数

**X** 是一个向量或矩阵。

## 详情

若 *X* 是向量，返回 X(n)\X(n-1) 的值。返回结果的第一个元素是 NULL。

若 *X* 是矩阵，在每列内进行上述计算，返回一个与 *X* 维度相同的矩阵。

## 例子

```
x=3 12 0 -5 32;
ratios x;
// output
[,4,0,,-6.4]

x=2 3 6 NULL 28 7;
ratios x;
// output
[,1.5,2,,,0.25]

m=matrix(100 105 109 112 108 116, 200 212 208 199 206 210);
m;
```

| #0 | #1 |
| --- | --- |
| 100 | 200 |
| 105 | 212 |
| 109 | 208 |
| 112 | 199 |
| 108 | 206 |
| 116 | 210 |

ratios(m);
