# asin

## 语法

`asin(X)`

## 参数

**X** 可以是标量、向量、矩阵或表。

## 详情

返回 *X* 的反正弦。

## 例子

```
asin(0 0.5 1);
// output
[0,0.523599,1.570796]
```
