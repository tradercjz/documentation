# atan

## 语法

`atan(X)`

## 参数

**X** 可以是标量、向量、矩阵或表。

## 详情

返回 *X* 的反正切。

## 例子

```
atan 0.000000 1.557408 -2.185040;
// output
[0,1,-1.141593]
```
