# atanh

## 语法

`atanh(X)`

## 参数

**X** 可以是标量、向量、矩阵或表。

## 详情

返回 *X* 的反双曲正切。

## 例子

```
atanh 0.000000 0.557408 -0.185040;
// output
[0,0.629065,-0.187196]
```
