# acosh

## 语法

`acosh(X)`

## 参数

**X** 可以是标量、向量、矩阵或表。

## 详情

返回 *X* 的反双曲余弦。

## 例子

```
acosh(1 2 3);
// output: [0,1.316958,1.762747]
```
