# acos

## 语法

`acos(X)`

## 参数

**X** 可以是标量、向量、矩阵或表。

## 详情

返回 *X* 的反余弦。

## 例子

```
acos(1.000000 0.540302 -0.416147);
// output: [0,1,2]
```
