# eq

## 语法

`eq(X, Y)` 或 `X==Y`

## 参数

**X** 和 **Y** 可以是标量、数据对、向量、矩阵或集合。如果 *X* 或 *Y*
其中一个是数据对、向量或矩阵，另一个必须是标量，或具有相同长度或维度的数据对、向量或矩阵。

## 详情

如果 *X* 和 *Y* 都不是集合，返回逐个元素比较 *X*==*Y* 的结果。

如果 *X* 和 *Y* 都是集合，则检查 *X* 和 *Y* 是否是同一个集合。

## 例子

```
1 2 3 == 2;
// output
[0,1,0]

1 2 3==0 2 4;
// output
[0,1,0]

1:2==1:6;
// output
1 : 0

m1=1..6$2:3;
m1;
```

集合操作：如果 *X*==*Y*，则 *X* 和 *Y* 是同一个集合。

```
x=set(4 6)
y=set(4 6 8);

x==y;
// output
0
x==x;
// output
1
```
