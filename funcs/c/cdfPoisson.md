# cdfPoisson

## 语法

`cdfPoisson(mean, X)`

## 参数

**mean** 是泊松分布的均值。

**X** 是整型标量或向量。

## 详情

返回泊松分布的累计分布函数的值。

## 例子

```
cdfPoisson(1, [-1, 0, 1, 2, 3]);
// output
[0, 0.367879, 0.735759, 0.919699, 0.981012]

cdfPoisson(1, [1, 3, 5, 7, 9]);
// output
[0.735759, 0.981012, 0.999406, 0.99999, 1]
```
