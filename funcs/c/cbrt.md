# cbrt

## 语法

`cbrt(X)`

## 参数

**X** 可以是标量、数据对、向量、矩阵或表。

## 详情

返回 *X* 的立方根。返回结果的数据类型为 DOUBLE 类型。

## 例子

```
cbrt(8);
// output
2

cbrt(8 12 16);
// output
[2,2.289428,2.519842]

cbrt(1..6$2:3);
```

| 0 | 1 | 2 |
| --- | --- | --- |
| 1 | 1.44225 | 1.709976 |
| 1.259921 | 1.587401 | 1.817121 |
