# cummin

## 语法

`cummin(X)`

参数说明和窗口计算规则请参考：[累计窗口系列（cum 系列）](../themes/cumFunctions.md)

## 详情

计算 *X* 元素的累计最小值。

## 例子

```
x = [7,4,5,NULL,9]
cummin(x);
// output
[7,4,4,4,4]

m = matrix(6 5 7 8 1, 3 9 4 2 10);
m;
```

| #0 | #1 |
| --- | --- |
| 6 | 3 |
| 5 | 9 |
| 7 | 4 |
| 8 | 2 |
| 1 | 10 |

```
cummin(m);
```

| #0 | #1 |
| --- | --- |
| 6 | 3 |
| 5 | 3 |
| 5 | 3 |
| 5 | 2 |
| 1 | 2 |

相关函数：[cummax](cummax.md), [min](../m/min.md)
