# cdfUniform

## 语法

`cdfUniform(lower, upper, X)`

## 参数

**lower** 和 **upper** 是数值型标量，表示连续均匀分布的下限和上限。

**X** 是数值型标量或向量。

## 详情

返回均匀分布的累计密度函数的值。

## 例子

```
cdfUniform(0.627, 2.31, [0.001, 0.5, 0.999]);
// output
[0, 0, 0.221034]

cdfUniform(0.627, 2.31, [0.1, 0.3, 0.5, 0.7, 0.9]);
// output
[0, 0, 0, 0.043375, 0.16221]
```
