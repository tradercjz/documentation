# cumPositiveStreak

## 语法

`cumPositiveStreak(X)`

参数说明和窗口计算规则请参考：[cumFunctions](../themes/cumFunctions.md)

## 详情

累计计算 *X* 中连续的正数之和的。

## 例子

```
x=1 0 -1 1 2 2 2 1 0 -1 0 2;

cumPositiveStreak x;
// output
[1,0,0,1,3,5,7,8,0,0,0,2]

m=matrix(1 0 -1 1 2 2 2 1 0 -1 0 2, -1 -2 -1 0 1 3 6 7 0 -1 -2 0);
m;
```

| #0 | #1 |
| --- | --- |
| 1 | -1 |
| 0 | -2 |
| -1 | -1 |

| 2 | 1 |
| 2 | 3 |
| 2 | 6 |
| 1 | 7 |
| 0 | 0 |
| -1 | -1 |
| 0 | -2 |
| 2 | 0 |

```
cumPositiveStreak(m);
```
