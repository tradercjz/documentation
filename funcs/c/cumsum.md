# cumsum

## 语法

`cumsum(X)`

参数说明和窗口计算规则请参考：[累计窗口系列（cum 系列）](../themes/cumFunctions.md)

## 详情

计算 *X* 元素的累计和。

## 例子

```
x=[2,3,4];
cumsum(x);
// output
[2,5,9]

m=matrix(1 2 3, 4 5 6);
m;
```

| #0 | #1 |
| --- | --- |
| 1 | 4 |
| 2 | 5 |
| 3 | 6 |

```
cumsum(m);
```

| #0 | #1 |
| --- | --- |
| 1 | 4 |
| 3 | 9 |
| 6 | 15 |

相关函数：[sum](../s/sum.md)
