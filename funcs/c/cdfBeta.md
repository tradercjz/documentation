# cdfBeta

## 语法

`cdfBeta(alpha, beta, X)`

## 参数

形状参数 **alpha** 和 **beta** 都是正数。

**X** 是数值型标量或向量。

## 详情

返回 Beta 分布的累计密度函数的值。

## 例子

```
cdfBeta(2.31, 0.627, [0.001, 0.5, 0.999]);
// output
[0, 0.116056, 0.976416]

cdfBeta(2.31, 0.627, [0.1, 0.3, 0.5, 0.7, 0.9]);
// output
[0.002451, 0.032995, 0.116056, 0.280532, 0.597694]
```
