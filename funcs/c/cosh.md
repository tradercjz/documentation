# cosh

## иҜӯжі•

`cosh(X)`

## еҸӮж•°

**X** еҸҜд»ҘжҳҜж ҮйҮҸгҖҒеҗ‘йҮҸжҲ–зҹ©йҳөгҖӮ

## иҜҰжғ…

иҝ”еӣһ *X* зҡ„еҸҢжӣІдҪҷејҰгҖӮ

## дҫӢеӯҗ

cosh 0 1 2;
// output
[1,1.543081,3.762196]
