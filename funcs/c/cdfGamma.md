# cdfGamma

## 语法

`cdfGamma(shape, scale, X)`

## 参数

形状参数 **shape** 是正数。

尺度参数 **scale** 是正数。

**X** 是数值型标量或向量。

## 详情

返回 Gamma 分布的累计密度函数的值。

## 例子

```
cdfGamma(2.31, 0.627, [0.001, 0.5, 0.999]);
// output
[0, 0.127367, 0.38032]

cdfGamma(2.31,0.627, [0.1, 0.3, 0.5, 0.7, 0.9]);
// output
[0.004754, 0.048388, 0.127367, 0.225351, 0.329391]
```
