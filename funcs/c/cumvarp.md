# cumvarp

## 语法

`cumvarp(X)`

参数说明和窗口计算规则请参考：[累计窗口系列（cum 系列）](../themes/cumFunctions.md)

## 详情

计算 *X* 元素的累计总体方差。

## 例子

```
cumvarp(1 2 3 NULL 4);
// output
[ , 0.25, 0.666666666666667, 0.666666666666667, 1.25]

m=matrix(1.1 3 5.0 7.5 9.2, 1 4.3 7.1 10.6 13.5);
m;
```

| col1 | col2 |
| --- | --- |
| 1.1 | 1 |
| 3 | 4.3 |
| 5 | 7.1 |
| 7.5 | 10.6 |
| 9.2 | 13.5 |

```
cumvarp(m);
```

| col1 | col2 |
| --- | --- |
| 0 | 0 |
| 0.9025 | 2.7225 |
| 2.5356 | 6.2156 |
| 5.6425 | 12.5025 |
| 8.5944 | 19.612 |

相关函数： [var](../v/varp.md)
