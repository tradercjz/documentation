# cdfBinomial

## 语法

`cdfBinomial(trials, p, X)`

## 参数

**trials** 是正整数。

**p** 是0到1之间的浮点数。

**trials** 和 **p** 是形状参数。

**X** 是数值型标量或向量。

## 详情

返回二项分布的累计密度函数的值。

## 例子

```
cdfBinomial(10, 0.1, [1, 5, 9]);
// output
[0.736099, 0.999853, 1]

cdfBinomial(12,0.627, [1, 3, 5, 7, 9]);
// output
[0.000154, 0.009085, 0.114844, 0.483879, 0.88373]
```
