# mul

## 语法

`mul(X, Y)` 或 `X*Y`

## 参数

**X** 和 **Y** 可以是标量、数据对、向量或矩阵。如果 *X* 和 *Y*
的其中一个是数据对、向量或矩阵，另一个必须是标量或具有相同长度或维度的数据对、向量或矩阵。

## 详情

逐个元素地返回 *X* 和 *Y* 的乘积。

## 例子

```
1:2*3;
// output
3 : 6

1:2*3:4;
// output
3 : 8

x=1 2 3;
x * 2;
// output
[2,4,6]

y=4 5 6;
x * y;
// output
[4,10,18]
```
