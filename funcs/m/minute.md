# minute

## 语法

`minute(X)`

## 详情

返回对应的分钟数。

## 参数

**X** 可以是整数标量或向量、时间标量或向量。

## 例子

```
minute();
```

返回：null

```
minute(1)
```

返回：00:01m

```
minute(now());
```

返回：16:02m
