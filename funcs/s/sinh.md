# sinh

## 语法

`sinh(X)`

## 参数

**X** 可以是标量、向量或矩阵。

## 详情

返回 *X* 的双曲正弦。

## 例子

```
sinh 1 2 3;
// output
[1.175201,3.62686,10.017875]
```
