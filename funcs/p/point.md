# point

## 语法

`point(X, Y)`

## 参数

**X** 和 **Y** 是数值型的标量、数据对、向量或矩阵，支持的数据类型为 INTEGRAL
类（COMPRESSED、INT128 除外）和 FLOATING 类。

## 详情

生成一个 POINT 类型的数据，用来存储坐标系中点的位置。

POINT 类型数据长度为16字节，其中低8位的数据存储于 *X* 中，高8位的数据存储于 *Y* 中。

## 例子

```
point(117.60972, 24.118418)
// output:
(117.60972, 24.118418)

point(1..5,6..10)
// output:
[(1, 6),(2, 7),(3, 8),(4, 9),(5, 10)]
```
