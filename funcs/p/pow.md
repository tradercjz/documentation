# pow

## 语法

`pow(X, Y)`

## 详情

返回以 *X* 中的元素为底，以 *Y* 中元素为指数计算得到的值。当 *X* 和 *Y*
为整数时，结果仍然是 DOUBLE 类型。

## 参数

**X** 和 **Y** 可以是标量、向量或矩阵。

## 例子

```
x=1 2 3;
pow(x,3);
//output:[1,8,27]

pow(3,x);
//output:[3,9,27]

y=4.5 5.5 6.5;
pow(x,y);
//output:[1,45.254834,1262.665039]

pow(y,x);
//output: [4.5,30.25,274.625]
```

```
m=1..10$2:5;
m;
```
