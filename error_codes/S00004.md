# S00004

## 错误代码

S00004

## 报错信息

Couldn't create a socket with error code <errno> RefId:S00004

## 错误原因

Server 创建套接字失败的原因可参考： [socket(2) - Linux manual page](https://man7.org/linux/man-pages/man2/socket.2.html)。

可能的原因包括：

1. 进程没有权限创建套接字。
2. 进程持有的文件描述符已达到系统限制。
3. 进程申请不到足够的内存作为套接字的缓存。

## 解决办法

根据错误原因，进一步分析排查问题。
