# mul(\*)

## 语法

`X * Y`

## 参数

**X** 和 **Y** 可以是标量、数据对、向量或矩阵。当X和Y都是向量或矩阵时，它们的长度或维度必须相同。

## 详情

返回X和Y逐个元素相乘的积。

## 例子

```
1:2*3;
// output
3 : 6

1:2*3:4;
// output
3 : 8

x=1 2 3;
x * 2;
// output
[2,4,6]

y=4 5 6;
x * y;
// output
[4,10,18]

m1=1..6$2:3;
m1;
```
