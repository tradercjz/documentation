# Python 到 DolphinDB 的函数映射

本篇介绍 Python 部分模块到 DolphinDB 函数库的不完全映射。

下文涉及的 python 模块如下：

* python built-in function
* numpy
* pandas
* scipy
* statsmodels
* sklearn
* TA-lib
